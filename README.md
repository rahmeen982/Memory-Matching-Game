#  Memory Matching Game
A simple, fun **Memory Matching Game** built with HTML, CSS, and jQuery.
Match all the cards to win and enjoy a smooth animated background and interactive gameplay!

---
##  Features

* Animated gradient background
* 4x2 card grid
* Click cards to reveal values
* Correct matches turn green
* Lock system prevents clicking multiple cards at once
* “Congratulations” message when all pairs are matched
* Restart button to play again

## How to Play

1. Click a card to reveal its value.
2. Click a second card to try to find a match.
3. If both cards match, they stay green.
4. If they don’t match, they hide automatically after 0.8 seconds.
5. Continue until all cards are matched.
6. Once all matches are found, a **Congratulations** message appears.
7. Use the **Restart** button to play again.
---
##  Technologies Used
* **HTML** – Structure of the game
* **CSS** – Styling & gradient animations
* **jQuery** – Interactive game logic
